实例

创建

每个 Vue 应用都是从创建一个 Vue 的根实例开始的：

创建一个实例时，可以传入选项对象，以创建用户想要的行为，可以参考官方的 API 文档 1。

可以包含数据、模板、挂载元素、方法、生命周期钩子等选项。

所有的 Vue 组件都是 Vue 实例，并且接受相同的选项对象 (一些根实例特有的选项除外)。

属性、方法

每个 Vue 实例都会代理其 data 对象里的所有属性，只有这些被代理的属性才是响应的：

除了 data 属性，Vue 实例还暴露了其他的 property（实例属性）和方法，这些都有 $ 前缀，以便和代理的 data 属性区分。

可以参考 API 文档 2，常用的实例属性：

  • vm.$data : 实例的数据对象

  • vm.$el : 实例的根 DOM 元素

  • vm.$patent : 当前实例的父实例
