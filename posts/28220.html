Docker

Ubuntu - 安装Docker

  1. 更新 APT，安装需要的 HTTPS 软件包和需要的证书

  2. 添加 GPG 秘钥

  3. 向 sources.list 中添加 Docker 软件源

  4. 更新 apt，安装 Docker

  5. 启动 Docker

  6. 当前不是 root 用户，需要建立 docker 用户组并加入

  7. 验证 Docker 是否安装成功

    显示以下内容则表示安装成功：

SVN

  1. 更新 apt，安装 svn

  2. 新建 svn文件夹 和 svn版本库，并且修改文件夹权限

    这是两个不同的文件夹，前者保存 svn 相关文件，启动、停止服务的命令都需要指向该文件夹；后者用于保存版本库相关信息，上传的文件都会保存在这里。

  3. 修改配置文件（1/3）：svnserve.conf 服务配置

  4. 修改配置文件（2/3）：password 配置用户和密码

    用户名为 user，密码为 userpassword。

  5. 修改配置文件（3/3）：authz 添加用户和用户组并设置权限

  6. 启动 svn 服务

  7. 查看是否成功运行

  8. 清理本地账号信息缓存

  9. 检出版本库

    这里需要在服务器地址后拼接步骤二中的版本库文件夹名称

Nginx

  1. 更新 apt，安装 nginx

  2. 查看是否成功

  3. 查询配置文件位置

  4. 启动后访问ip地址，是否出现 nginx 标识

    如果没有出现截图中的内容，需要在服务器安全组规则中添加 80 端口，并且在防火墙放行
