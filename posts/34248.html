变量

  1. 使用 var 关键字定义，忘记使用关键字直接赋值也是有效的，还同时可以声明多个变量。

  2. 变量的数据类型没有限制，可以随时更改

  3. 如果只声明变量没有赋值，则该变量的值为特殊值 undefined，表示无定义

变量提升

JavaScript 引擎的工作方式是，先解析代码，获取所有被声明的变量，然后再一行一行地运行。
这造成的结果，就是所有的变量的声明语句，都会被提升到代码的头部，这就叫做变量提升（hoisting）。

变量名规则

  1. javascript 对大小写敏感，所以 a 和 A 是两个不同变量。

  2. 变量名开头必须使用如下的字符：

    • 任意 Unicode 字符（包括英文或其他语言的字母）
    • 美元符号（$）
    • 下划线（_）

  3. 第二个及之后的字符规则如下：

    • 任意 Unicode 字符（包括英文或其他语言的字母）
    • 美元符号（$）
    • 下划线（_）
    • 数字（0-9）

  4. 中文也是合法的标识符，可以当作变量名使用

  5. 不允许使用保留字，例如：new，private，for 等
